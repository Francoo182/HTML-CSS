Formas de enlazar CSS

Tenemos 3 formas distintas de darle estilo a la pagina

  1. La primera es darle directamente a la etiqueta(Mala practica)
  2. La segunda es dar dentro del mismo archivo HTML utilizando un style>/style>(Tambien mala practica)
  3. La tercera y mas importante Enlazar a un archivo externo de CSS utilizando Link>/Link> que dentro tiene los atributos de rel="stylesheet"(Esto le dice que el archivo que vamos a vincular o relacionar tiene una relacion de Estilo con el documento) y un href
