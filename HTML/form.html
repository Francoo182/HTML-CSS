INPUTS

Recibir un Input del usuario, sea el que sea esto se hace atravez de un form cuantos input type:"TipoDeInput" vamos a tener muchos tipos de Input, botones, texto, etc... Los inputs mas conocidos son los de

  • text
  • email
  • password
  • checkbox
  • radio
  • file
  • submit
  • date

Atributos mas usados en inputs HTML

  • *Required* = Es obligatorio este campo(Funcionalidad), se le puede agregar min lentgh =3 para decir que almenos necesitamos 3 caracteres
  • *name*= Le asignamos un nombre a la variable input para usarla en JS o en CSS(Interno para mi)
  • *placeholder* = Un texto que desaparece cuando escribes, por ejemplo, "Aqui escribe tu nombre"
